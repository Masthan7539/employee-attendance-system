## 📌 Project Description
Employee Attendance Management System is a Python-based application used to manage and track employee attendance efficiently. It includes user authentication, attendance marking, and database integration.

---

## 🚀 Features
- User Login System
- Employee Attendance Marking
- Attendance Record Storage
- Secure Database Integration
- Simple and User-Friendly Interface

---

## 🛠️ Technologies Used
- Python
- SQLAlchemy (ORM)
- SQLite / MySQL / SQL Server
- HTML/CSS (if frontend included)

---

## 📂 Project Structure
- backend/ → Contains main logic and Python files
- frontend/ → UI files (if available)
- db_setup.py → Database setup script
- main.py → Main application file
